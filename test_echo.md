# 回环测试步骤

## 当前配置
- 模式：RUN_MODE_ECHO_TEST = 1（USART2回环测试）
- 串口：USART2 (PA2/PA3)
- RS485控制：PA4
- LED指示：PB1
- 波特率：115200

## 测试步骤

1. **编译项目**
   - 确保 app_config.h 中 RUN_MODE_ECHO_TEST = 1

2. **烧录到目标板**

3. **观察启动**
   - PB1 LED应该闪烁3次（启动指示）

4. **测试回环**
   - 连接串口工具到USART2 (PA2/PA3)
   - 波特率：115200
   - 发送任意数据
   - 应该原样返回

## LED指示说明
- 启动时闪3次：系统初始化成功
- 正常运行时有心跳闪烁

## 故障排查
1. 检查IDLE中断是否触发
2. 检查DMA是否正确配置
3. 检查RS485方向控制（PA4）
